HTML5

HTML structural

HTML byggs genom en rad element eller taggar. Taggarna öppnas och stängs och innehållet skrivs mellan öppnad och stängd tagg. Taggar kan ha egenskaper som anpassar dessa, vidare kan man sätta id och classer på alla taggar för att göra det möjligt att nå dessa mer precist från t.ex javascript eller css
